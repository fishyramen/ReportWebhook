# ReportWebhook

A lightweight Paper plugin that allows players to report others using `/report (user) (reason)` and sends the report to a Discord webhook using a clean embed format.

---

## Features

- /report <player> <reason>
- Sends formatted Discord embed to webhook
- No in-game coordinates are sent
- Includes reporter, reported, reason, world, and UTC time
- Configurable webhook URL
- Lightweight and async-safe
